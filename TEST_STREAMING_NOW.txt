❌ TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   Problema: Testo appare tutto insieme
   ↓
   Soluzione: 
   - Apri DevTools → Network → /api/chat → Response
   - Se dati incrementali → Frontend OK, problema backend
   - Se tutto insieme → Backend/proxy bufferizza

   ───────────────────────────────────────────────────────────────

   Problema: Cursore non visibile
   ↓
   Soluzione:
   - Hard refresh: Ctrl + Shift + R
   - Check console errori JavaScript
   - Verify CSS caricato

   Problema: Latenza alta (> 1 sec)
   ↓
   Soluzione:
   - Verifica console: ⚡ First chunk > 1000ms
   - È problema backend (cold start?), non frontend

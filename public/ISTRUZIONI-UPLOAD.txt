TROUBLESHOOTING
================================================================================

‚ùå PROBLEMA: Pagina bianca o errori 404
‚úÖ SOLUZIONE:

   - Verifica che il nome del file sia esattamente "index.html" (minuscolo)


‚ùå PROBLEMA: Stili CSS non caricati (pagina senza colori/grafica)
‚úÖ SOLUZIONE:

‚ùå PROBLEMA: JavaScript non funziona (bottoni non rispondono)
‚úÖ SOLUZIONE:

‚ùå PROBLEMA: Errore CORS quando invii messaggi
‚úÖ SOLUZIONE:
   - Questo significa che il backend su Cloud Run non accetta richieste dal tuo dominio
   - Devi configurare CORS sul backend con questo comando:

   gcloud run services update vanda-chatbot --region europe-west1 \
     --update-env-vars "ALLOWED_ORIGINS=https://www.agentika.io,https://agentika.io"


‚ùå PROBLEMA: Chat non riceve risposte
‚úÖ SOLUZIONE:
   - Apri DevTools (F12) ‚Üí Network ‚Üí invia un messaggio
   - Cerca la richiesta POST a /api/chat
   - Se vedi errori 401/403 ‚Üí problema di autenticazione API
   - Se vedi errori 500 ‚Üí problema backend
   - Se vedi errori CORS ‚Üí vedi soluzione sopra
